movemenoreg.vbs病毒的处理

发表于 2016-12-31   |   分类于 瞎玩   |     |   本文总阅读量 次

近期在学校周围火速窜起一种将U盘文件变为一个以U盘名字命名的快捷方式的病毒。
貌似传染速度非常快，以下是解决方式及病毒原理的基本分析

0x01 最简单的解决方式

一个台湾大神根据这种病毒制作了一个USBVirusKiller软件，直接可以进行清理
百度云链接：链接: https://pan.baidu.com/s/1dFxaz3j 密码: kh7p
Tips: 需要.NetFramework 4.0及以上环境，而一般win7默认是3.5，win8及以后的window版本可以直接使用。win7用户可以安装4.0，或者直接跳到0x03处理。

0x02 病毒特征及其感染范围

vbs病毒：利用vb(virtual basic)的脚本语言vbs编写的病毒程序，具有简单易懂，可修改性强的特点。顾其感染范围也限制在Window平台的电脑上。

中毒后的U盘目录结构：

[未隐藏] U盘名.lnk 快捷方式 病毒启动点

[隐 藏] _ 文件夹 包含U盘里的原始文件

[隐 藏] WindowsServices 文件夹 包含以下病毒流程文件

  • install.vbs
  • helper.vbs
  • movemenoreg.vbs

0x03 一般处理步骤

1.[插入U盘后没有点击.lnk文件]只要在window或者mac上直接勾选显示隐藏文件，将WindowsServices文件夹和.lnk文件直接删除即可去除病毒，同时将_文件夹下的内容拷贝出来。
2.[插入U盘后点击了.lnk文件]处理步骤分以下几步：
  2.1 先将U盘内病毒文件删除。
  2.2 window +r 输出msconfig,将名为helper.vbs的气动项关闭。
  2.3 ctl+Alt+Del 调出任务管理器，将一个或多个wscript.exe进程终止。
  2.4 window + r 输入%AppData%，在打开的文件夹里勾选显示隐藏文件，会看到WindowsServices文件夹将该文件夹内容删除。
  2.5 window + r 输入shell:startup ，将里面的helper的快捷方式删除。
  2.6 至此处理完成
